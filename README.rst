Install
------------------------------------------------------------------------------

``learn_temporal`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install learn-temporal

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade learn-temporal
